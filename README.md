# nz-phishing-selective-detection
NZ-localized phishing and smishing detection with selective abstention. Combines a localized feature set (IRD, NZ Post, banks, te reo cues, .govt.nz lookalikes) with classical baselines, fine-tuned transformers, and LLM-as-classifier (Gemini), under a class-conditional deferral layer for human review. Independent research.
