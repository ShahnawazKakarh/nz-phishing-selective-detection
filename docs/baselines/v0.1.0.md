## Honest findings

1. **Ceiling effect on synthetic NZ subset.** Both variants score 1.0 F1 on
   `source=nz_synth`. The v0.1.0 synthetic templates are textually distinctive
   enough that TF-IDF n-grams alone separate them perfectly, leaving no room
   for NZ-localized features to add measurable lift. This invalidates the
   *aggregate* NZ-features-vs-ablation comparison on this dataset.
2. **NZ features add marginal cost on UCI SMS.** The aggregate F1 gap
   (0.9730 vs 0.9778, with-NZ vs without-NZ) is driven entirely by `uci_sms`,
   where NZ cues are mostly absent and contribute weak noise.
3. **TF-IDF + LR is a strong global-spam baseline.** Both variants exceed 0.95
   F1 / 0.998 AUC on UCI SMS, consistent with the literature.

## Reproduce

```bash
python scripts/collect_uci_sms.py
python scripts/synth_nz_lures.py --seed 42 --n-per-class-channel 300
nzphish-prepare --seed 42
nzphish-train --config configs/tfidf_lr.yaml --seed 42
nzphish-train --config configs/tfidf_lr_no_nz.yaml --seed 42
nzphish-eval-stratified --run-dir results/runs/<run_with_nz>    --split test
nzphish-eval-stratified --run-dir results/runs/<run_without_nz> --split test
```

## v0.2.0 plan (addresses v0.1.0 ceiling)

1. **Harder NZ benchmark.** Paraphrased synthetic lures (LLM-rewritten) and
   adversarial NZ samples where lure keywords are subtly embedded in
   otherwise-normal text. Designed so TF-IDF alone cannot saturate.
2. **Real NZ data.** Nazario phishing emails filtered for NZ-relevant brands,
   plus a Gemini-graded NZ relevance score on top of PhishTank URL captures.
3. **Two stronger classifier families.** Fine-tuned DistilBERT and Gemini 2.5
   Flash as LLM-as-classifier (zero-shot + few-shot), so the NZ-features
   contribution can be measured against capacity-matched baselines, not just
   linear models.
4. **Selective deferral.** Class-conditional confidence calibration with
   coverage / selective-F1 reporting at multiple operating points.
